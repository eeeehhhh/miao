So What is This About?

There is continuing need to show the power of CSS. The Zen Garden aims to excite, inspire ,and encourage participation. To begin, view some of the existing desgins in the style sheet into this very page. The HTMLremains the same,the thing that has changed is the external CSS allows complete and total control over the style of a hypertext document. The only way this can be illustrated in a way that gets people truly be, once the reins are placed in the hands of those able to create beauty from structure,Designers and coders alike have contribute to the beauty of the web; we can always push it further.

Participation

Stong visual desgin has always been our focus. You are modifying this page, so strong CSSnovices can use them as starting points. Please see the CSS Resource Guide for advanced tutorials and tips on working with CSS.

You may modify thr style sheet in any way you wish, but not the HTML. This may seem daunting at first if you've never worked this way before,but follow the listed links to learn more, and use the sample files as a guide.

Downlond the sample HTML and CSS to work on a copy locally. once you have completed your masterpiece (and please,don't submit half-finished work) upload your CSS file to a web server under your control. Send us a linkto an archive of that file and all associated assets, and if we choose to use it we will download it and place it on our server.

Benefits

Why participat? For recongnition inspisation, and a resource we can all refer to showing people how amazing CSS really can be. This site serves as equal parts inspiration for those working on the web today ,learning tool for those who will be tommorrow, and gallery of future techniques we can all look forward to.

Requirement

Where possible, we would like to see mostly CSS1 & 2 usage.CSS 3 & 4 should be limited to widely-supported elements only, or strong fallbacks should be provied. The CSS Zen Garden is about functional, practical CSSand not the latest bleeding-edge triks viewable by 2% of the browing public. The only real requirement we have is that your CSSvalidates.

Luckily,designing this way shows how well various browsers have implemented CSS by now. When sticking to the guidelines you should see fairly consistent results across most modern browsers. Due to the sheer number of user agent on the web these days - especially when you factor in mobile - pixel-perfect layouts may not be possbile across every platform. That's okay, but do test in as many as you can. Your design should work in at least IE9+ and the latest Chorme, Firefox,ios and Android browsers (run by over 90% of the population).

We ask that you submit original artwork. please respect copyright laws. Please keep objectionable material to a minmum, and try to incorporate unique an interesting visual themes to your work. We're well past the point of needing another garden-related desgin.
